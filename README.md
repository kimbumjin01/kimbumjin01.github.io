# kimbumjin01.github.io

김범진의 개인 허브, 포트폴리오, 공부 노트 연결용 GitHub Pages 사이트입니다.

## 구성

- `index.html`: Home, Projects, Notes, About, Resume 섹션
- `assets/styles.css`: 반응형 스타일
- `assets/script.js`: 연도 표시용 작은 스크립트
- `assets/hero-workspace.png`: 사이트 첫 화면 이미지

## 배포 방법

1. GitHub에서 `kimbumjin01.github.io` 이름의 public repository를 만듭니다.
2. 이 폴더의 파일들을 repository 루트에 올립니다.
3. repository의 `Settings > Pages`에서 source를 `Deploy from a branch`, branch를 `main`, folder를 `/root`로 설정합니다.
4. 잠시 후 `https://kimbumjin01.github.io`에서 확인합니다.

## math-notes 연결

현재 버튼은 `https://kimbumjin01.github.io/math-notes/`로 연결되어 있습니다.
math-notes 주소가 다르면 `index.html`에서 `math-notes` 링크만 변경하면 됩니다.

## PDF 이력서 연결

공개용 PDF를 연결하려면 `assets/resume.pdf` 파일을 추가한 뒤 `index.html`의 Resume 섹션 버튼을 PDF 링크로 바꾸면 됩니다.
